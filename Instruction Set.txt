FORMAT: <OPCODE> <DESTINATION>, <SOURCE> 
          	        Operation    Code (Hex)
Load A:     	    LOD A	     10
Store A:    	    STO A        11
Load B:             LOD B        12
Store B:            STO B        13
Add A:      	    ADD          20
Subtract A: 	    SUB          21
Add With Carry A:   ADC	         22
Jump:               JMP          30
Jump If Zero:       JZ           31
Jump If Carry:      JC           32
Jump If Not Zero:   JNZ	         33
Jump If Not Carry:  JNC		     34
Move A, B           MOV A, B     40     (Only requires 1 byte, no address required)
Move B, A           MOV B, A     41
Halt:               HLT          FF



CODE FOR 16 BIT ADDITION
    LOD A,   [low byte 1]
    ADD A,   [low byte 2]
    STO [low byte sum], A
    LOD A,   [high byte 1]
    ADC A,   [high byte 2]
    STO [high byte sum], A
